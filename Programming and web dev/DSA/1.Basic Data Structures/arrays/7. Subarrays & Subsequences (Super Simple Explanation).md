# **SUBARRAY (Easy Explanation)**

ðŸ‘‰ **Continuous** part of the array.  
ðŸ‘‰ You cannot skip elements.

### **Example Problem**

Find **all subarrays** of:

`[1, 2, 3]`

### **Explanation**

We pick a start point and then extend continuously.

### **Output**

`1 1 2 1 3 2 2 3 3`

### âœ… **C++ Code**

`#include <iostream> using namespace std;  int main() {     int arr[] = {1, 2, 3};     int n = 3;      for (int start = 0; start < n; start++) {         for (int end = start; end < n; end++) {             for (int i = start; i <= end; i++)                 cout << arr[i] << " ";             cout << endl;         }     } }`
