# ✅ **When to use Two Pointers (very simple)**

Use **Two Pointers** when:

### **1. You need to check from both sides**

Like checking if a word is a **palindrome**.

### **2. You need to find two numbers that match a condition**

Like two numbers that **add up to a target** (in a **sorted** array).

### **3. You need to move through a list fast without using two loops**

Like finding a **subarray/substring** (sliding window).

### **4. You want to move or filter elements in place**

Like **removing duplicates** or **moving zeros**.





# ✅ **QUESTION 1 — Remove Duplicates from Sorted Array**

**(Most famous Two Pointer question)**

### ❓ Problem

Given a **sorted array**, remove duplicates **in-place** and return the count of unique elements.

### ✅ C++ Code

```cpp
int removeDuplicates(int nums[], int n) {
    if (n == 0) return 0;

    int i = 0;
    for (int j = 1; j < n; j++) {
        if (nums[j] != nums[i]) {
            i++;
            nums[i] = nums[j];
        }
    }
    return i + 1;
}

int main() {
    int nums[] = {1,1,2,2,3,3,4};
    int n = sizeof(nums)/sizeof(nums[0]);

    int k = removeDuplicates(nums, n);
    cout << "Unique Count: " << k << "\\n";

    cout << "Unique elements: ";
    for (int i = 0; i < k; i++) cout << nums[i] << " ";
}

```
