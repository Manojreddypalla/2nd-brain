# Software Engineering

> The discipline of designing, developing, testing, deploying, and maintaining software systems.

---

# Purpose

Software Engineering focuses on building reliable, scalable, maintainable software.

Software can include:

- Websites
- Mobile Apps
- Desktop Applications
- Games
- Cloud Services
- AI Applications
- Operating Systems
- Databases
- Embedded Systems

---
