## Unit 2: Basic Commands and Directory Hierarchy — Introduction

Linux is a command-line–driven operating system where users interact with the system through commands executed in a shell. While graphical interfaces exist, the command line provides powerful, efficient, and precise control over files, directories, processes, and system resources. Understanding basic Linux commands is the first step toward becoming comfortable with Linux administration, development, cybersecurity, and cloud technologies.

A Linux system organizes all files and resources using a hierarchical directory structure that starts from a single root directory (`/`). Unlike Windows, which uses separate drives such as `C:` and `D:`, ==Linux treats everything—including files, devices, and storage—as part of one unified directory tree. Each directory has a specific purpose, such as storing user files, system configurations, executable programs, and device information.==

In this unit, we will learn how to navigate the Linux file system, manage files and directories, understand common Linux commands, and explore the standard directory hierarchy. These concepts form the foundation for working effectively in any Linux environment.

### Key Learning Outcomes

After completing this unit, you should be able to:

- Understand the Linux command-line interface (CLI).
    
- Use basic Linux commands for navigation and file management.
    
- Create, copy, move, rename, and delete files and directories.
    
- View and manipulate file contents.
    
- Understand the Linux directory hierarchy.
    
- Identify the purpose of important directories such as `/bin`, `/etc`, `/home`, `/usr`, and `/var`.
    
- Navigate the filesystem efficiently using absolute and relative paths.
