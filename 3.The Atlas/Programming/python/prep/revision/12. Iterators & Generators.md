# 1. Create an Iterator

```python
nums = [10, 20, 30]

it = iter(nums)
```

**Explanation:**

- Converts an iterable (list) into an iterator.
    
- Iterator remembers its current position.
    

---

# 2. Get Next Element

```python
print(next(it))
```

Output

```
10
```

Again

```python
print(next(it))
```

Output

```
20
```

**Explanation**

- Returns one element.
    
- Moves to the next position.
    

---

# 3. StopIteration

```python
nums = [1]

it = iter(nums)

print(next(it))
print(next(it))
```

Output

```
1
StopIteration
```

**Explanation**

- Happens when iterator has no more elements.
    

---

# 4. Iterator in for Loop

```python
nums = [1,2,3]

for i in nums:
    print(i)
```

Internally Python does

```python
it = iter(nums)

while True:
    try:
        i = next(it)
        print(i)
    except StopIteration:
        break
```

**Explanation**

- Every `for` loop uses an iterator internally.
    

---

# 5. Generator Function

```python
def numbers():
    yield 1
    yield 2
    yield 3
```

Create generator

```python
g = numbers()
```

**Explanation**

- `yield` creates a generator.
    
- Doesn't execute immediately.
    

---

# 6. next() on Generator

```python
print(next(g))
```

Output

```
1
```

Again

```python
print(next(g))
```

Output

```
2
```

**Explanation**

- Generator resumes from where it paused.
    

---

# 7. return vs yield

### return

```python
def test():
    return 10
```

Output

```python
print(test())
```

```
10
```

Function ends immediately.

---

### yield

```python
def test():
    yield 10
```

Output

```python
print(test())
```

```
<generator object ...>
```

Need

```python
g = test()

print(next(g))
```

Output

```
10
```

---

# 8. Generator Expression

List

```python
nums = [x*x for x in range(5)]
```

Generator

```python
nums = (x*x for x in range(5))
```

**Difference**

List

```
Stores everything
```

Generator

```
Creates one value at a time
```

---

# 9. Iterate over Generator

```python
def count():

    yield 1
    yield 2
    yield 3

for i in count():
    print(i)
```

Output

```
1
2
3
```

---

# 10. Memory Difference

List

```python
nums = [i for i in range(1000000)]
```

- Stores 1 million integers in memory.
    

Generator

```python
nums = (i for i in range(1000000))
```

- Stores almost nothing.
    
- Produces values only when requested.

# ⭐ Interview Summary

|Concept|Remember|
|---|---|
|Iterable|Can be looped over (`list`, `tuple`, `set`, `dict`, `str`)|
|Iterator|Created by `iter()`|
|`next()`|Returns next element|
|`StopIteration`|Iterator finished|
|Generator|Function with `yield`|
|`yield`|Pauses and resumes function|
|`return`|Ends function|
|Generator Expression|`(x for x in range(...))`|
|List Comprehension|`[x for x in range(...)]`|

### One-line intuition to remember forever

- **Iterable** → "A collection you can loop over."
    
- **Iterator** → "A cursor that walks through the collection one item at a time."
    
- **Generator** → "A special iterator that _creates_ items only when you ask for them."
