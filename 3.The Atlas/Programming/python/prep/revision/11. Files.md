# 11. File Handling

## What is File Handling?

**Definition:** Reading from and writing to files stored on your computer.

---

# 1. Open a File

```python
file = open("data.txt", "r")
```

**Definition:** Opens a file.

---

# Modes

|Mode|Meaning|
|---|---|
|`"r"`|Read|
|`"w"`|Write (creates/overwrites)|
|`"a"`|Append|
|`"x"`|Create new file|
|`"rb"`|Read binary|
|`"wb"`|Write binary|

---

# 2. Read Entire File

```python
file = open("data.txt", "r")

print(file.read())

file.close()
```

---

# 3. Read One Line

```python
file = open("data.txt", "r")

print(file.readline())

file.close()
```

---

# 4. Read All Lines

```python
file = open("data.txt", "r")

print(file.readlines())

file.close()
```

Output

```python
['Hello\n', 'World\n']
```

---

# 5. Write to File

```python
file = open("data.txt", "w")

file.write("Hello World")

file.close()
```

⚠ `"w"` replaces old content.

---

# 6. Append

```python
file = open("data.txt", "a")

file.write("\nPython")

file.close()
```

Adds to the end.

---

# 7. Best Practice ⭐⭐⭐⭐⭐

Use `with`.

```python
with open("data.txt", "r") as file:
    print(file.read())
```

**Definition:** Automatically closes the file.

No need for

```python
file.close()
```

---

# 8. Check if File Exists

```python
import os

print(os.path.exists("data.txt"))
```

---

# 9. Exception Handling

```python
try:
    with open("data.txt", "r") as file:
        print(file.read())

except FileNotFoundError:
    print("File not found")
```

# 10. CSV Files (Very Important)

Read CSV

```python
import pandas as pd

df = pd.read_csv("students.csv")

print(df)
```

---

Write CSV

```python
df.to_csv("output.csv", index=False)
```

---

# 11. Excel Files

```python
import pandas as pd

df = pd.read_excel("students.xlsx")
```

---

# 12. JSON Files

Read

```python
import json

with open("data.json", "r") as file:
    data = json.load(file)

print(data)
```

Write

```python
import json

data = {"name":"Retro","age":22}

with open("data.json", "w") as file:
    json.dump(data, file)
```

# Most Common Interview MCQ

```python
with open("file.txt","r") as file:
    data = file.read()

print(file.closed)
```

Output

```python
True
```

Because `with` automatically closes the file.

# Why Data Science Uses File Handling

Almost every dataset starts as a file:

```
Excel
JSON
Text
```

Pandas simply reads these files into a **DataFrame**.

