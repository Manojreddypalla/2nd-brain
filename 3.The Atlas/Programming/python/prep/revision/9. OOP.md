# 1. Class

**Definition:** A blueprint for creating objects.

```python
class Student:
    pass
```

---

# 2. Object

**Definition:** An instance of a class.

```python
class Student:
    pass

s1 = Student()
```

---

# 3. Constructor (`__init__`)

**Definition:** Automatically called when an object is created.

```python
class Student:
    def __init__(self, name):
        self.name = name

s1 = Student("Retro")
print(s1.name)
```

---

# 4. `self`

**Definition:** Refers to the current object (similar to `this` in C++).

```python
class Student:
    def __init__(self, name):
        self.name = name

    def display(self):
        print(self.name)

s1 = Student("Retro")
s1.display()
```

---

# 5. Instance Variable

**Definition:** Variable unique to each object.

```python
class Student:
    def __init__(self, name):
        self.name = name

s1 = Student("Retro")
s2 = Student("Alex")
```

---

# 6. Class Variable

**Definition:** Shared by all objects.

```python
class Student:
    college = "ABC College"

print(Student.college)
```

---

# 7. Inheritance

**Definition:** Child class acquires properties and methods of the parent class.

```python
class Animal:
    def speak(self):
        print("Animal speaks")

class Dog(Animal):
    pass

d = Dog()
d.speak()
```

---

# 8. Method Overriding

**Definition:** Child class provides its own implementation of a parent method.

```python
class Animal:
    def speak(self):
        print("Animal")

class Dog(Animal):
    def speak(self):
        print("Bark")

Dog().speak()
```

---

# 9. Polymorphism

**Definition:** Same method name, different behavior.

```python
class Dog:
    def speak(self):
        print("Bark")

class Cat:
    def speak(self):
        print("Meow")

animals = [Dog(), Cat()]

for animal in animals:
    animal.speak()
```

---

# 10. Encapsulation

**Definition:** Restrict direct access to data.

```python
class Student:
    def __init__(self):
        self.__age = 22

s = Student()

# print(s.__age)   # Error
```

---

# 11. Abstraction

**Definition:** Hide implementation details and show only essential functionality.

```python
from abc import ABC, abstractmethod

class Animal(ABC):

    @abstractmethod
    def sound(self):
        pass

class Dog(Animal):
    def sound(self):
        print("Bark")

Dog().sound()
```

---

# 12. Magic Method (`__str__`)

**Definition:** Defines how an object is displayed as a string.

```python
class Student:
    def __init__(self, name):
        self.name = name

    def __str__(self):
        return self.name

s = Student("Retro")
print(s)
```
