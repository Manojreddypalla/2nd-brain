# 1. `len()`

**Definition:** Returns the number of elements.

```python

```

Output

```text

```

# 19. `enumerate()` ⭐⭐⭐⭐⭐

Returns **index and value**.

```python
names = ["A","B","C"]

for index, value in enumerate(names):
    print(index, value)
```

Output

```text
0 A
1 B
2 C
```

---

# 20. `zip()` ⭐⭐⭐⭐⭐

Combines multiple iterables.

```python
name = ["A","B"]
age = [20,21]

print(list(zip(name, age)))
```

Output

```python
[('A',20), ('B',21)]
```

---

# 21. `map()` ⭐⭐⭐⭐⭐

Applies a function to every element.

```python
nums = [1,2,3]

result = list(map(lambda x: x*x, nums))

print(result)
```

Output

```python
[1,4,9]
```

---

# 22. `filter()` ⭐⭐⭐⭐⭐

Filters elements based on a condition.

```python
nums = [1,2,3,4,5]

result = list(filter(lambda x: x%2==0, nums))

print(result)
```

Output

```python
[2,4]
```
