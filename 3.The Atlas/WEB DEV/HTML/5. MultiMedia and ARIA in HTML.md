# 1. Multimedia in HTML

HTML is not only for text and headings.

It can also handle:

- Images
- Videos
- Audio
- Accessibility features (ARIA)

---

# 2. Image Tag `<img>`

Used to display images in a webpage.

## Syntax

```
<imgsrc="image.png"alt="description">
```

## Example

```
<imgsrc="hc.png"alt="profile photo"width="200">
```

## Important Attributes

|Attribute|Purpose|
|---|---|
|`src`|Image path|
|`alt`|Alternative text if image fails|
|`width`|Width of image|
|`height`|Height of image|
|`title`|Tooltip text|

---

## What is `alt`?

If image cannot load, browser shows alternative text.

### Example

```
<imgsrc="wrongname.png"alt="profile photo">
```

If image missing → `"profile photo"` appears.

---

## What is `title`?

Shows small tooltip when mouse hovers.

```
<imgsrc="hc.png"title="Hitesh">
```

---

# 3. Figure Tag `<figure>`

Semantic way to group image + caption.

## Structure

```
<figure>
<imgsrc="hc.png"alt="image"width="200">
<figcaption>Profile Photo</figcaption>
</figure>
```

---

## Why use `<figure>`?

Because image + caption become one meaningful unit.

Useful for:

- Blogs
- Articles
- Documentation
- News websites

# 5. Video Tag `<video>`

Used to display videos.

## Basic Syntax

```
<videosrc="video.mp4"></video>
```

But video will NOT play properly without controls.

---

# 6. Video Controls

Add `controls` attribute.

```
<videosrc="video.mp4"controls></video>
```

Now user gets:

- Play
- Pause
- Volume
- Fullscreen

---

# 7. Video Width

```
<videosrc="video.mp4"controlswidth="400"></video>
```

---

# 8. Video Control List

Used to disable some controls.

## Example

```
<videocontrolscontrolsList="nodownload">
```

Possible values:

|Value|Meaning|
|---|---|
|`nodownload`|Removes download option|
|`nofullscreen`|Removes fullscreen|
|`noremoteplayback`|Disable remote playback|

---

# 9. Audio Tag `<audio>`

Used to add audio/music.

## Example

```
<audiosrc="song.mp3"controls></audio>
```

---

# 10. Semantic HTML

Semantic tags give meaning to content.

Examples:

|Tag|Meaning|
|---|---|
|`<header>`|Top section|
|`<footer>`|Bottom section|
|`<article>`|Article content|
|`<figure>`|Media with caption|

---

# 11. ARIA Tags

ARIA = Accessible Rich Internet Applications

Used for accessibility.

Helps:

- Screen readers
- Keyboard navigation
- Speech recognition software

---

# 12. Common ARIA Attributes

|Attribute|Purpose|
|---|---|
|`role`|Defines element role|
|`aria-label`|Gives readable label|
|`aria-hidden`|Hide from screen readers|

---

## Example

```
<buttonaria-label="Close Menu">
    X
</button>
```

---

# 13. Why Accessibility Matters

Important for:

- Banking websites
- Government websites
- Disabled users
- Better usability

Questions accessibility engineers ask:

- Can website be used with keyboard only?
- Can screen readers understand it?
- Can speech software use it?

# Quick Interview/Exam Questions

## What is the purpose of `alt` attribute?

Used to show alternative text when image fails to load.

---

## Difference between `<img>` and `<figure>`?

`<img>` → only image

`<figure>` → image + caption as one semantic unit

---

## Why use `controls` in video tag?

To give play/pause/volume controls.

---

## What is ARIA?

Accessible Rich Internet Applications.

Used to improve accessibility.
