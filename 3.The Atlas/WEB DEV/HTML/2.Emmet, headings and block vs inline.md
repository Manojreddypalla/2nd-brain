# 📘 HTML – Lecture Notes (Headings, Lists, Images, Links, Block vs Inline)

---

# 1️⃣ HTML Basics

### 🔹 HTML = HyperText Markup Language

- Used to structure content on the web.
- Not for styling (CSS does that).
- Not for logic (JavaScript does that).

---

# 2️⃣ Emmet (VS Code Shortcut Tool)

### 🔹 What is Emmet?

- Built-in tool in VS Code.
- Helps generate HTML quickly.

### 🔹 Common Shortcuts

```
! + Tab        → Full HTML boilerplate
h1             → <h1></h1>
h1>lorem6      → h1 with 6 words
ul>li*3        → 3 list items
div>ul>li      → nested structure
```

### 🔹 Lorem Generator

```
lorem10 → 10 random words
```

---

# 3️⃣ Headings

### 🔹 6 Levels

```
<h1> to <h6>
```

### 🔹 Important Concept

- NOT about size.
- It represents importance.
- h1 = most important
- h6 = least important

⚠ Styling (size/color) is done using CSS.

---

# 4️⃣ Paragraph

```
<p></p>
```

- Used for text content.
- Block-level element.

---

# 5️⃣ HTML Terminology

### 🔹 Element

Complete structure:

```
<h1>Hello</h1>
```

### 🔹 Tag

```
<h1> → opening tag
</h1> → closing tag
```

### 🔹 Attribute

Properties inside tags:

```
<img src="image.png" width="300">
```

Common attributes:

- src
- href
- width
- title
- alt

---

# 6️⃣ Images

```
<img src="path" alt="description">
```

### 🔹 Important:

- No closing tag.
- src → image path
- alt → accessibility text
- width → controls size

### 🔹 Path Types

- Relative: `./images/img.png`
- Same folder: `image.png`

---

# 7️⃣ Anchor Tag (Links)

```
<a href="<https://example.com>">Click</a>
```

### 🔹 Types of Links

1. External link
2. Internal page link
3. Email link

```
<a href="mailto:test@gmail.com">
```

1. Phone link

```
<a href="tel:9876543210">
```

---

# 8️⃣ Lists

## 🔹 Unordered List (UL)

```
<ul>
  <li>Item</li>
</ul>
```

- Bullet points
- Order not important

## 🔹 Ordered List (OL)

```
<ol>
  <li>Item</li>
</ol>
```

- Numbered
- Order matters

# 9️⃣ Block vs Inline Elements

Very Important Concept 🔥

---

## 🔹 Block-Level Elements

- Take full width (left to right)
- Start on new line

### Examples:

```
h1 – h6
p
ul
ol
li
div
```

---

## 🔹 Inline Elements

- Stay inside the line
- Don’t take full width

### Examples:

```
a
img
strong
em
b
i
```
