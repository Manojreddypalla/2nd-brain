# 📌 HTML – Tables & Forms Notes

## 🔹 1. Two Main Jobs of HTML

HTML mainly does **2 things**:

1. 📄 **Display Data**
2. 📥 **Collect Data from Users**

---

# 🟢 PART 1: TABLES (Displaying Tabular Data)

## 📌 Why Tables?

Used to display:

- Student data
- Research data
- Reports
- Pricing tables
- Scoreboards

---

## 🧱 Basic Table Structure

```html
<table>
  <tr>
    <th>Heading 1</th>
    <th>Heading 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>
```

---

## 🧩 Important Table Tags

|Tag|Meaning|Purpose|
|---|---|---|
|`<table>`|Table|Main container|
|`<tr>`|Table Row|Creates row|
|`<th>`|Table Heading|Bold heading cell|
|`<td>`|Table Data|Normal cell|

---

## 📌 Important Points

- Each `<tr>` = one row
- Number of `<td>` should match number of `<th>`
- Browser auto-adjusts if missing data
- Tables look plain until styled with **CSS**

---

## 🧠 Example: Student Table

```html
<table>
  <tr>
    <th>Name</th>
    <th>Email</th>
    <th>DOB</th>
  </tr>
  <tr>
    <td>Manoj</td>
    <td>manoj@mail.com</td>
    <td>02/02/2004</td>
  </tr>
</table>
```

---

# 🟢 PART 2: FORMS (Taking User Input)

## 📌 What is a Form?

A form is used to:

- Login
- Register
- Submit data
- Upload files
- Collect feedback

---

## 🧱 Basic Form Structure

```html
<form>
   <!-- Input fields -->
</form>
```

Form alone does nothing.

You must add **input fields**.

---

# 🟵 INPUT TAG

```html
<input type="text">
```

Most important tag inside forms.

---

# 🟡 Common Input Types

|Type|Purpose|
|---|---|
|text|Normal text|
|password|Hidden input|
|email|Email validation|
|url|URL validation|
|number|Only numbers|
|date|Date picker|
|month|Month picker|
|color|Color picker|
|file|File upload|
|checkbox|Multiple selection|
|radio|Single selection|
|submit|Submit form|
|reset|Reset form|

---

# 🟢 Label Tag

```html
<label for="email">Email:</label>
<input type="email" name="email">
```

### Why label?

- Improves accessibility
- Connects text with input
- Good practice

---

# 🟢 Example: Login Form

```html
<form>
  <label>Email:</label>
  <input type="email" name="email">

  <label>Password:</label>
  <input type="password" name="password">

  <button type="submit">Login</button>
  <button type="reset">Reset</button>
</form>
```

---

# 🟢 Special Input Features

## 1️⃣ Placeholder

```html
<input type="text" placeholder="Enter your name">
```

- Shows hint text
- Disappears when typing

---

## 2️⃣ Email Validation

```html
<input type="email">
```

Browser automatically checks:

- Must contain `@`
- Must follow email format

---

## 3️⃣ Textarea (Large Input)

```html
<textarea rows="5" cols="30"></textarea>
```

Used for:

- Comments
- Feedback
- Messages

---

## 4️⃣ Checkbox

```html
<input type="checkbox"> Cricket
<input type="checkbox"> Football
```

User can select multiple.

---

## 5️⃣ Radio Button

```html
<input type="radio" name="gender"> Male
<input type="radio" name="gender"> Female
```

User can select only one.

---

## 6️⃣ Select Dropdown

```html
<select>
  <option>Cricket</option>
  <option>Football</option>
</select>
```

Creates dropdown menu.

# 🟢 Interview-Level Summary

### HTML Responsibilities:

1. Structure content
2. Display data
3. Collect data

### Tables:

- Used for structured data
- Built using table → tr → th/td

### Forms:

- Used for input collection
- Built using form + input
- Many input types available
- Validation built-in for some types

---

# 🟢 Very Important Concept

HTML = Structure

CSS = Styling

JavaScript = Functionality

Example:

- HTML → Login form structure
- CSS → Make it beautiful
- JS → Check login credentials
