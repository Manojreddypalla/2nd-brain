# 📌 HTML Semantics & Building a Full Webpage – Notes

---

# 🟢 1. What is Semantic HTML?

### 🔹 Meaning:

Semantic HTML = Using meaningful tags to structure your webpage.

Instead of using only `<div>`, we now use special tags that describe the content.

---

## ❌ Old Way (Non-semantic)

```html
<div class="header"></div>
<div class="footer"></div>
```

Problem:

- No meaning
- Bad for screen readers
- Poor accessibility

---

## ✅ Modern Way (Semantic HTML5)

```html
<header></header>
<nav></nav>
<main></main>
<article></article>
<section></section>
<footer></footer>
```

These tags:

- Improve accessibility
- Help screen readers
- Improve SEO
- Make code readable

---

# 🟢 2. Basic Structure of a Website

Most websites follow this structure:

```
Header
   └── Navigation
Main Content
   ├── Article
   ├── Sections
Footer
```

---

# 🟢 3. Important Semantic Tags

|Tag|Purpose|
|---|---|
|`<header>`|Top section of page|
|`<nav>`|Navigation links|
|`<main>`|Main content|
|`<article>`|Independent content|
|`<section>`|Section inside content|
|`<footer>`|Bottom section|
|`<address>`|Contact information|
|`<time>`|Time/date|
|`<code>`|Code block|
|`<hgroup>`|Group of headings|

---

# 🟢 4. Example Full Webpage Structure

```html
<!DOCTYPE html>
<html>
<head>
  <title>Gym Website</title>
</head>

<body>

<header>
  <h1>Learn to Create Website</h1>
  <h2>Build Your Own Websites</h2>

  <nav>
    <ul>
      <li><a href="#">Home</a></li>
      <li><a href="#">About</a></li>
      <li><a href="#">Contact</a></li>
    </ul>
  </nav>
</header>

<article>
  <section>
    <p>This is first section.</p>
  </section>

  <section>
    <code>console.log("Hello")</code>
  </section>
</article>

<footer>
  <address>Hyderabad, India</address>
  &copy; 2024
</footer>

</body>
</html>
```

---

# 🟢 5. Special Semantic Elements Explained

---

## 🔹 `<nav>`

Used for:

- Menu
- Navigation links
- Sidebar links

Example:

```html
<nav>
  <a href="#">Home</a>
</nav>
```

---

## 🔹 `<article>`

Used for:

- Blog posts
- News articles
- Independent content

---

## 🔹 `<section>`

Used to divide content into logical parts.

Think:

Article = Book

Section = Chapter

---

## 🔹 `<time>`

```html
<time datetime="2024-08-02">August 2</time>
```

Used for:

- Dates
- Deadlines
- Events

Good for search engines.

---

## 🔹 `<code>`

```html
<code>console.log("Hello")</code>
```

Used to display programming code.

---

## 🔹 `<address>`

Used for:

- Author info
- Contact details
- Company address

---

# 🟢 6. Special Characters in HTML

To add symbols:

Start with `&` (ampersand)

Example:

```html
&copy;
```

Displays:

©

Other examples:

- `&lt;` → <
- `&gt;` → >
- `&amp;` → &

---

# 🟢 7. Meta Tags (Inside `<head>`)

Example:

```html
<meta charset="UTF-8">
```

Purpose:

- Defines character encoding
- UTF-8 supports most languages

Other meta tags:

- Viewport
- SEO info
- Description

---

# 🟢 8. Global Attributes

These work on almost every tag:

|Attribute|Purpose|
|---|---|
|id|Unique identifier|
|class|Group styling|
|title|Tooltip text|

Used mainly in:

- CSS
- JavaScript

---

# 🟢 9. Important Concept

Semantic tags DO NOT change layout.

They are:

- Container tags
- Like `<div>`
- But meaningful

CSS decides:

- Position
- Colors
- Size
- Design
