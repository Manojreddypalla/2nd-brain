# 📘 HTML – Beginner Notes (From This Video)

## 1. What is HTML?

**HTML = HyperText Markup Language**

- It is **not a programming language**
- It is a **markup language**
- Used to **structure content** on the web
- Example: headings, paragraphs, images, tables, links, etc.

HTML tells the browser:

> “This is a heading”, “This is a paragraph”, “This is a button”

---

## 2. Why HTML was created?

Originally, scientists wanted to share **research papers** online:

- Headings
- Paragraphs
- Tables
- Links

So HTML was created to **format content** properly on the web.

---

## 3. HTML Uses Tags

HTML works using **tags**:

### Example:

```html
<h1>Hello World</h1>

```

- `<h1>` → Opening tag
- `</h1>` → Closing tag
- `Hello World` → Content

Most HTML tags are **containers** (they open and close).

---

## 4. Structure of an HTML Document

Every HTML page follows a **fixed structure** so it works in all browsers.

### Basic Structure:

```
HTML
 ├── Head (Metadata)
 └──Body (Visiblecontent)

```

---

## 5. What is `<head>`?

The `<head>` contains **metadata** (information about the page).

Examples:

- Page title
- Charset
- Description
- SEO info
- Icons

Not visible on the page.

---

## 6. What is `<body>`?

The `<body>` contains **visible content**:

- Headings
- Paragraphs
- Images
- Buttons
- Forms

---

## 7. Boilerplate Code (Default HTML Template)

This is the **minimum required HTML structure**.

```html
<!DOCTYPE html>
<htmllang="en">
<head>
<title>My First Webpage</title>
</head>
<body>

</body>
</html>

```

---

## 8. What is `<!DOCTYPE html>`?

It tells the browser:

> This document uses HTML5

Always write it at the top.

---

## 9. Creating Your First HTML Page

### Step 1: Create a file

Name it:

```
index.html

```

Why `index.html`?

- Servers automatically load this file by default
