# 1. Why Learn JavaScript?

- One of the most popular programming languages.
- Used by millions of developers.
- Can build:
    - Websites
    - Web Applications
    - Backend Servers
    - Desktop Apps
    - Mobile Apps
    - Browser Extensions

**Key Idea:**

> Learn one language and use it almost everywhere.

---

# 2. Before JavaScript

Websites were mostly:

- HTML → Structure
- CSS → Styling

They looked good but didn't do much.

Example:

- Static portfolio website
- Static landing page

These are called **web pages**, not web applications.

---

# 3. Role of JavaScript

### HTML

🏠 Structure

### CSS

🎨 Design

### JavaScript

⚡ Behavior

Examples:

- Button click
- Form validation
- Dropdown menus
- To-Do app
- Dynamic content

**Formula:**

HTML + CSS + JavaScript = Interactive Website

---

# 4. How JavaScript Was Originally Used

Earlier JavaScript could run only inside browsers.

Example:

```
console.log("Hello World");
```

Run using:

```
Right Click → Inspect → Console
```

Because browsers contained the JavaScript engine.

---

# 5. Programming Language Execution

Every language needs:

```
Code ↓Compiler/Interpreter ↓Output
```

Example:

```
JavaScript File      ↓ JavaScript Engine      ↓ Behavior/Output
```

Without an engine, code is just text.

---

# 6. V8 Engine

Created by:

Google

### What is V8?

- High-performance JavaScript engine
- Written in C++
- Open source

Purpose:

```
JavaScript Code       ↓      V8       ↓   Machine Execution
```

---

# 7. Node.js

### What is Node.js?

Node.js uses the V8 engine outside the browser.

Before:

```
JavaScript → Browser Only
```

After Node.js:

```
JavaScript → Anywhere
```

Can create:

- Servers
- APIs
- CLI Tools
- Backend Applications

**Definition:**

> Node.js is a JavaScript runtime environment.

---

# 8. Why Node.js Was Revolutionary

Developers could now use:

### Same Language Everywhere

Frontend:

```
JavaScript
```

Backend:

```
JavaScript
```

No need to switch languages.
