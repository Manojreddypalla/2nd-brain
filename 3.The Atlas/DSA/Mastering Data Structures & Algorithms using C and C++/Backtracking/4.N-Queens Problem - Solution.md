### Problem

Place **N queens on an N×N chessboard** such that:

```
No two queens attack each other
```

Queens should not be in the same:

- Row
- Column
- Diagonal

---

# Main Idea

Place queens **row by row**.

```
Row 1 → Place Q1Row 2 → Place Q2Row 3 → Place Q3...
```

For each row:

```
Try every column
```

If position is safe:

```
Place QueenMove to next row
```

If no safe position exists:

```
Go backMove previous queenTry another column
```

This is **Backtracking**.

---

# Mental Picture

```
Place Queen      ↓Safe?      ↓Yes → Next Row      ↓No → Next ColumnNo Column Works?      ↓Backtrack
```

---

# Bounding Condition

Before placing a queen check:

```
Same Column ?  ❌Same Diagonal ? ❌
```

If yes:

```
Reject immediately
```

---

# Example (4 Queens)

Solution:

```
2 4 1 3
```

Meaning:

```
Q1 → Column 2Q2 → Column 4Q3 → Column 1Q4 → Column 3
```

---

# Pattern Recognition

N-Queens is just:

```
Choose Column↓Check Safe↓Place Queen↓Go Next Row↓Undo If Needed
```

### One-Line Memory Trick

```
N-Queens = Place one queen per row; if stuck, go back and move the previous queen.
```
