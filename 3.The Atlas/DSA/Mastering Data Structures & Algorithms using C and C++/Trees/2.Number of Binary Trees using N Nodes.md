# Number of Binary Trees (Catalan Numbers)

## Problem

Given **N nodes**, how many different binary trees can be formed?

---

# Unlabeled Nodes

Unlabeled nodes mean only the **shape** matters.

Example:

```
o   o   o
```

No values are stored inside the nodes.

---

## Examples

### N = 3

Number of different binary trees:

```
T(3) = 5
```

### N = 4

Number of different binary trees:

```
T(4) = 14
```

---

# Catalan Number Formula

The number of binary trees with N nodes is given by the Catalan Number:

$$
T(N)=\frac{1}{N+1}\binom{2N}{N}
$$

Equivalent form:

$$
T(N)=\frac{(2N)!}{(N+1)!N!}
$$

---

## Catalan Number Table

| N | Catalan Number |
|---|---|
| 0 | 1 |
| 1 | 1 |
| 2 | 2 |
| 3 | 5 |
| 4 | 14 |
| 5 | 42 |
| 6 | 132 |
| 7 | 429 |

---

# Intuition

Suppose we choose one node as the root.

Remaining nodes must be distributed between:

- Left Subtree
- Right Subtree

For every possible split:

```
Number of Left Trees
×
Number of Right Trees
```

Add all possibilities.

This leads to the Catalan recurrence.

---

# Recursive Formula

$$
T(N)=\sum_{i=1}^{N} T(i-1)\times T(N-i)
$$

---

## Example: T(6)

$$
T(6)=T(0)T(5)+T(1)T(4)+T(2)T(3)+T(3)T(2)+T(4)T(1)+T(5)T(0)
$$

Substituting values:

$$
=1(42)+1(14)+2(5)+5(2)+14(1)+42(1)
$$

$$
=42+14+10+10+14+42
$$

$$
=132
$$

# Trees with Maximum Height

Maximum height occurs when the tree becomes completely skewed.

Example:

```
o
|
o
|
o
|
o
```

or

```
o
 \
  o
   \
    o
```

---

## Formula

For N nodes:

$$
2^{N-1}
$$

---

## Examples

### N = 3

$$
2^{3-1}=4
$$

### N = 4

$$
2^{4-1}=8
$$

### N = 5

$$
2^{5-1}=16
$$

# Labeled Nodes

Nodes contain values:

```
A  B  C
```

Now both:

- Shape matters
- Arrangement of labels matters

---

## Label Arrangements

For N labels:

$$
N!
$$

ways to arrange labels.

Example:

For 3 labels:

$$
3! = 6
$$

Possible arrangements:

```
ABC
ACB
BAC
BCA
CAB
CBA
```

---

# Labeled Binary Tree Formula

Unlabeled tree count:

$$
\frac{1}{N+1}\binom{2N}{N}
$$

Label arrangements:

$$
N!
$$

Therefore:

$$
\text{Labeled Trees}
=
\frac{1}{N+1}\binom{2N}{N}\times N!
$$
