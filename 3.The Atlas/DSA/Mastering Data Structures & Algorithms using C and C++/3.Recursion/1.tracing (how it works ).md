# Recursion — Short Notes

Recursion = **a function calling itself**.

```cpp
void fun(int n)
{
    if(n > 0)   // base condition
    {
        cout << n << " ";
        fun(n - 1);   // recursive call
    }
}
```

---

# Core Idea

A recursive function has:

1. **Base Case** → stopping condition
2. **Recursive Case** → function calls itself

Without base case → infinite recursion → stack overflow.

---

# Mental Model

Think of recursion like:

- Entering rooms inside rooms
- Or stretching a rubber band

Two phases happen:

1. **Calling Phase** → going deeper
2. **Returning Phase** → coming back

---

# Important Concept — Stack

Every function call gets stored in the **call stack**.

Example:

```cpp
fun(3)
```

Stack internally becomes:

```
fun(3)
fun(2)
fun(1)
fun(0)
```

Then returns backward:

```
fun(0) returns
fun(1) returns
fun(2) returns
fun(3) returns
```

---

# Example 1 — Print During Calling Phase

```cpp
#include <iostream>
using namespace std;

void fun(int n)
{
    if(n > 0)
    {
        cout << n << " ";
        fun(n - 1);
    }
}

int main()
{
    fun(3);
}
```

## Output

```
3 2 1
```

## Why?

Print happens BEFORE recursive call.

Flow:

```
print 3
print 2
print 1
```

---

# Example 2 — Print During Returning Phase

```cpp
#include <iostream>
using namespace std;

void fun(int n)
{
    if(n > 0)
    {
        fun(n - 1);
        cout << n << " ";
    }
}

int main()
{
    fun(3);
}
```

## Output

```
1 2 3
```

## Why?

Printing happens AFTER recursion finishes.

So values print while stack is returning.

# Dry Run

```cpp
fun(3)
```

Internally:

```
fun(3)
  fun(2)
    fun(1)
      fun(0)
```

Then return back.

---

# Base Condition

Very important.

```cpp
if(n > 0)
```

Stops recursion at:

```cpp
n == 0
```

Without this:

```cpp
fun(n-1);
```

keeps calling forever.

---

# Recursive Thinking Pattern

When solving recursion:

## Ask 3 things

### 1. Smallest problem?

```cpp
fun(1)
```

---

### 2. Base condition?

```cpp
if(n == 0)
    return;
```

---

### 3. How does current problem reduce?

```cpp
fun(n-1)
```

Smaller version of same problem.

---

# Important Observation

Recursion = Divide into smaller self-similar problems.

Examples later:

- Factorial
- Fibonacci
- Tree traversal
- DFS
- Backtracking
- DP

All use same idea.

---

# Factorial Example

```cpp
#include <iostream>
using namespace std;

int fact(int n)
{
    if(n == 0)
        return 1;

    return n * fact(n - 1);
}

int main()
{
    cout << fact(5);
}
```

## Internally

```
fact(5)
= 5 * fact(4)
= 5 * 4 * fact(3)
...
= 5 * 4 * 3 * 2 * 1
```

Output:

```
120
```

---

# Fibonacci Example

```cpp
#include <iostream>
using namespace std;

int fib(int n)
{
    if(n <= 1)
        return n;

    return fib(n-1) + fib(n-2);
}

int main()
{
    cout << fib(6);
}
```

Output:

```
8
```

---

# Memory Understanding

Each recursive call stores:

- local variables
- parameters
- return address

inside stack memory.

So recursion uses more memory than loops.

---

# Time Complexity Idea

If function calls itself once:

```cpp
fun(n-1)
```

Usually:

```
O(n)
```

If function calls twice:

```cpp
fib(n-1) + fib(n-2)
```

Can become exponential.

---

# Recursion vs Loop

|Loop|Recursion|
|---|---|
|Uses iteration|Uses function calls|
|Uses less memory|Uses stack memory|
|Faster usually|Cleaner sometimes|
|Good for simple repetition|Good for trees/graphs/divide problems|

---

# Most Important Pattern

You’ll later notice:

```
Linked List → recursive traversal
Tree → naturally recursive
Graph DFS → recursive
Backtracking → recursive state exploration
DP → optimized recursion
```

Recursion is basically:

> “Let smaller version solve the smaller problem.”

# Tiny Practice Codes

## Sum of N numbers

```cpp
#include <iostream>
using namespace std;

int sum(int n)
{
    if(n == 0)
        return 0;

    return n + sum(n-1);
}

int main()
{
    cout << sum(5);
}
```

Output:

```
15
```

## Reverse String

```cpp
#include <iostream>
using namespace std;

void reverse(string s, int i)
{
    if(i < 0)
        return;

    cout << s[i];
    reverse(s, i-1);
}

int main()
{
    string s = "hello";
    reverse(s, s.length()-1);
}
```

Output:

```
olleh
```
