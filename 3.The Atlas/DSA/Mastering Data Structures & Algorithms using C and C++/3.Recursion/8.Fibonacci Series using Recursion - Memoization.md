# Fibonacci Using Recursion — Exam Notes

---

# 1. Fibonacci Series

A Fibonacci series is a sequence in which each term is obtained by adding the previous two terms.

```text
0 1 1 2 3 5 8 13 21 ...
```

Example:

```text
0 + 1 = 1
1 + 1 = 2
1 + 2 = 3
2 + 3 = 5
3 + 5 = 8
```

---

# 2. Recursive Definition

The Fibonacci sequence can be defined recursively as:

[  
Fib(n)=Fib(n-1)+Fib(n-2)  
]

### Base Cases

[  
Fib(0)=0  
]

[  
Fib(1)=1  
]

---

# 3. Recursive Function

```cpp
int fib(int n)
{
    if(n <= 1)
        return n;

    return fib(n-1) + fib(n-2);
}
```

---

# 4. Working of Recursion

For:

```cpp
fib(5)
```

Recursion Tree:

```text
                 fib(5)
               /        \
          fib(4)       fib(3)
         /     \       /     \
     fib(3) fib(2) fib(2) fib(1)
      / \      / \    / \
 fib(2)fib(1)fib(1)fib(0)fib(1)fib(0)
```

Result:

```text
fib(5) = 5
```

---

# 5. Problem with Recursive Fibonacci

Observe the recursion tree:

```text
fib(3) calculated multiple times
fib(2) calculated multiple times
fib(1) calculated multiple times
```

The same subproblems are solved repeatedly.

This is called:

## Excessive Recursion

or

## Overlapping Subproblems

---

# 6. Time Complexity

Recursive relation:

```text
fib(n)
→ fib(n-1)
→ fib(n-2)
```

Each call creates two more calls.

Time Complexity:

[  
O(2^n)  
]

Space Complexity:

[  
O(n)  
]

because of recursion stack.

---

# 7. Need for Dynamic Programming

Problem:

```text
Same Fibonacci values are calculated repeatedly.
```

Example:

```text
fib(3)
fib(2)
fib(1)
```

are computed many times.

This wastes time.

Solution:

```text
Calculate once
Store result
Reuse result
```

This technique is called:

## Memoization

---

# 8. Fibonacci Using Memoization (DP)

```cpp
int dp[100];

int fib(int n)
{
    if(n <= 1)
        return n;

    if(dp[n] != -1)
        return dp[n];

    dp[n] = fib(n-1) + fib(n-2);

    return dp[n];
}
```

---

# 9. Advantages of Memoization

### Before DP

```text
Time Complexity = O(2^n)
```

### After DP

```text
Time Complexity = O(n)
```

Large improvement in performance.

---

# 10. Relationship Between Recursion and DP

```text
Recursion
      ↓
Repeated Subproblems
      ↓
Store Answers
      ↓
Memoization
      ↓
Dynamic Programming
```

Dynamic Programming is essentially:

```text
Recursion + Memory
```

# Iterative Fibonacci (Most Efficient Basic Version)

```cpp
int fib(int n)
{
    if(n <= 1)
        return n;

    int a = 0;
    int b = 1;
    int c;

    for(int i = 2; i <= n; i++)
    {
        c = a + b;
        a = b;
        b = c;
    }

    return b;
}
```

### Complexity

```text
Time  : O(n)
Space : O(1)
```
