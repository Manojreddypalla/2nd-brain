# 1. Sum of Natural Numbers using Recursion

## Idea

To find:


1 + 2 + 3 + ... + n  


Break the problem into a smaller one:


sum(n)=n+sum(n-1)  


### Base Case

When n = 0


sum(0)=0  


### Recursion Tree (n=4)

```text
sum(4)
= 4 + sum(3)
= 4 + 3 + sum(2)
= 4 + 3 + 2 + sum(1)
= 4 + 3 + 2 + 1 + sum(0)
= 10
```

## Code

```cpp
#include <iostream>
using namespace std;

int sum(int n)
{
    if(n == 0)
        return 0;

    return n + sum(n - 1);
}

int main()
{
    int n = 5;
    cout << sum(n);

    return 0;
}
```

Time Complexity: O(n)

Space Complexity: O(n)

# 2. Factorial using Recursion

## Idea

Factorial:

 
n! = n \times (n-1)!  


Example:

 
5! = 5 \times 4!  

### Base Case

 
0! = 1  


### Recursion Tree

```text
fact(5)
= 5 * fact(4)
= 5 * 4 * fact(3)
= 5 * 4 * 3 * fact(2)
= 5 * 4 * 3 * 2 * fact(1)
= 120
```

## Code

```cpp
#include <iostream>
using namespace std;

int fact(int n)
{
    if(n == 0 || n == 1)
        return 1;

    return n * fact(n - 1);
}

int main()
{
    cout << fact(5);

    return 0;
}
```

Time Complexity: O(n)

Space Complexity: O(n)

# 3. Power using Recursion

## Idea

To calculate:


m^n  


Use:


power(m,n)=m \times power(m,n-1)  


### Base Case

  
m^0 = 1  


### Example

```text
2^4

= 2 * power(2,3)
= 2 * 2 * power(2,2)
= 2 * 2 * 2 * power(2,1)
= 2 * 2 * 2 * 2 * power(2,0)

=16
```

## Code

```cpp
#include <iostream>
using namespace std;

int power(int m, int n)
{
    if(n == 0)
        return 1;

    return m * power(m, n - 1);
}

int main()
{
    cout << power(2,4);

    return 0;
}
```

Time Complexity: O(n)

Space Complexity: O(n)

# 4. Taylor Series using Recursion

## Formula
![[Pasted image 20260607122044.png]]
    

### Recursive Relation

Each term depends on previous factorial and power values.

## Code

```cpp
#include <iostream>
using namespace std;

double taylor(int x, int n)
{
    static double p = 1;
    static double f = 1;

    double r;

    if(n == 0)
        return 1;

    r = taylor(x, n - 1);

    p = p * x;
    f = f * n;

    return r + p / f;
}

int main()
{
    cout << taylor(1,10);

    return 0;
}
```

### Example

```text
x = 1

e¹ ≈ 2.71828
```

Time Complexity: O(n)

Space Complexity: O(n)

# 5. Taylor Series using Horner's Rule

## Why Horner's Rule?

Normal Taylor Series repeatedly calculates powers and factorials.

Horner's Rule rewrites:

![[Pasted image 20260607122119.png]]

This reduces calculations significantly.

---

## Recursive Formula

```text
s = 1 + (x/n) * s
```

starting from the last term.

## Code

```cpp
#include <iostream>
using namespace std;

double e(int x, int n)
{
    static double s = 1;

    if(n == 0)
        return s;

    s = 1 + (double)x / n * s;

    return e(x, n - 1);
}

int main()
{
    cout << e(1,10);

    return 0;
}
```

Time Complexity: O(n)

Space Complexity: O(n)

# Quick Exam Revision Table

|Problem|Recursive Formula|Base Case|
|---|---|---|
|Sum of N Numbers|sum(n)=n+sum(n−1)|n=0|
|Factorial|fact(n)=n×fact(n−1)|n=0 or 1|
|Power|power(m,n)=m×power(m,n−1)|n=0|
|Taylor Series|previous term + current term|n=0|
|Horner's Rule|s=1+(x/n)s|n=0|

### Pattern to Remember

Almost every recursion problem has:

```cpp
if(base_case)
    return answer;

return work + recursive_call(smaller_problem);
```

Think:

1. What is the smallest problem? (Base Case)
    
2. How can the current problem be expressed using a smaller version of itself?
    
3. Return the result.
    

That's the core idea behind all five recursion programs.
