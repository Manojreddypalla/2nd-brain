# Fibonacci using Dynamic Programming (Memoization)

## Text

fib(5)

1. Check if n <= 1
2. Check if dp[n] already has answer
3. If not:
   dp[n] = fib(n-1) + fib(n-2)
4. Store result in dp[]
5. Return dp[n]

Memoization avoids recomputing the same subproblems.
