### Mental Model

Every node carries a **valid range** with it.

- Root can be anything:
    - `(-∞, +∞)`
- Left child must be smaller than parent:
    - `(min, parent)`
- Right child must be larger than parent:
    - `(parent, max)`

So every recursive call asks:

> "Is this node allowed to exist within this range?"

```cpp
class Solution {
public:

    bool check(TreeNode* root, long long mn, long long mx) {

        if (root==nullptr)
            return true;

        if (root->val <= mn || root->val >= mx)
            return false;

        return check(root->left, mn, root->val) &&
               check(root->right, root->val, mx);
    }

    bool isValidBST(TreeNode* root) {
```
