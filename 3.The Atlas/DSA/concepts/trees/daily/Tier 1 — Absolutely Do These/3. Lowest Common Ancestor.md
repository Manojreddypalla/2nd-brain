### Lowest Common Ancestor in BST — Quick Notes

**BST Property**

```
left < root < right
```

**Idea**  
At each node:

```
Both p and q smaller -> go leftBoth p and q greater -> go rightOtherwise -> current node is LCA
```

**Why?**  
The first node where `p` and `q` are on different sides (or one equals the current node) is the **lowest common ancestor**.

**Pattern**

```
p,q < root  => leftp,q > root  => rightelse         => root is answer
```

**Time Complexity:** `O(H)`  
(H = height of BST)

**Space Complexity:** `O(H)` recursive, `O(1)` iterative

**One-line Memory Trick:**

> Keep moving toward both nodes; when they stop being on the same side, you've found the LCA.


```cpp 
/**

 * Definition for a binary tree node.

 * struct TreeNode {

 *     int val;

 *     TreeNode *left;

 *     TreeNode *right;

 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}

 * };

 */

  

class Solution {

public:

    TreeNode* lowestCommonAncestor(TreeNode* root, TreeNode* p, TreeNode* q) {

        if(p->val<root->val&&q->val<root->val)

        {

            return lowestCommonAncestor(root->left,p,q);

        }

  

        else if (p->val>root->val&&q->val>root->val)

        {

            return lowestCommonAncestor(root->right,p,q);

        }

  

        return root;

    }

};
```
