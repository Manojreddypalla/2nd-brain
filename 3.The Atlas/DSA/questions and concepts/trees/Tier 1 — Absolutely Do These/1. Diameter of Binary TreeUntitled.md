### Diameter of Binary Tree — Quick Notes

- Diameter = **longest path between any two nodes**.
- Count **edges**, not nodes.
- Path **does not need to pass through the root**.

### Key Formula

At every node:

```
diameterThroughNode = leftHeight + rightHeight
```

### Recursion

```
Subtree returns -> HeightGlobal variable -> Maximum Diameter
```

### Steps

```
1. Get left subtree height2. Get right subtree height3. Update ans = max(ans, left + right)4. Return 1 + max(left, right)
```

### Pattern

```
Return Height ↑Update Diameter Globally
```

### Remember

```
Parent needs Height, not Diameter.That's why dfs() returns Height.
```

This single line is the entire problem:

```
ans = max(ans, leftHeight + rightHeight);return 1 + max(leftHeight, rightHeight);
```


```cpp
class Solution {
public:
    int ans = 0;

    int height(TreeNode* root) {
        if (root == nullptr)
            return 0;

        int l = height(root->left);
        int r = height(root->right);

        ans = max(ans, l + r);

        return 1 + max(l, r);
    }

    int diameterOfBinaryTree(TreeNode* root) {
        height(root);
        return ans;
    }
};
```
