# tools

## **TOOLS in smolagents**

### üîπ What Are Tools?

- Tools are **functions or interfaces** that an **LLM agent** can call to perform specific actions.
- Each tool has:
    - **Name:** Identifier of the tool
    - **Description:** Explains what the tool does
    - **Inputs:** Arguments with types and descriptions
    - **Output:** Type of the returned result

üìò *Example:*

| Component | Example |
| --- | --- |
| Name | `web_search` |
| Description | Searches the web for queries |
| Input | `query: string` |
| Output | Search results as a string |

---

## üß∞ **Tool Creation Methods**

Tools in smolagents can be created in **two main ways:**

---

### üß† **1. Using the `@tool` Decorator** *(For Simple Tools)*

- Recommended for small, simple functions.
- smolagents automatically extracts metadata (name, docstring, types).

‚úÖ **Example:**

```python
from smolagents import CodeAgent, InferenceClientModel, tool

@tool
def catering_service_tool(query: str) -> str:
    """Returns the highest-rated catering service in Gotham City.
    Args:
        query: search term for finding catering services."""
    services = {"Gotham Catering Co.": 4.9, "Wayne Manor Catering": 4.8}
    return max(services, key=services.get)

agent = CodeAgent(tools=[catering_service_tool], model=InferenceClientModel())
agent.run("Find best catering service in Gotham City")

```

üß© *Key Benefits:*

- Simpler syntax
- Automatically readable by LLM
- Great for quick, single-purpose functions

---

### ‚öôÔ∏è **2. Subclassing the `Tool` Class** *(For Complex Tools)*

- Used when a tool requires advanced logic, multiple inputs, or metadata.

‚úÖ **Example:**

```python
from smolagents import Tool

class SuperheroPartyThemeTool(Tool):
    name = "superhero_party_theme_generator"
    description = "Suggests creative superhero-themed party ideas."
    inputs = {"category": {"type": "string", "description": "Type of party"}}
    output_type = "string"

    def forward(self, category: str):
        themes = {"classic heroes": "Justice League Gala", "villain masquerade": "Gotham Rogues' Ball"}
        return themes.get(category.lower(), "Try another theme.")

```

Then:

```python
agent = CodeAgent(tools=[SuperheroPartyThemeTool()], model=InferenceClientModel())
agent.run("Give me a 'villain masquerade' superhero theme idea.")

```

üß† *Advantages:*

- Full control over input/output
- Ideal for structured or multi-step tools
- Easier to maintain complex logic

## üß© **Default Toolbox (Built-in Tools)**

smolagents provides several pre-made tools:

| Tool | Description |
| --- | --- |
| `PythonInterpreterTool` | Executes Python code |
| `FinalAnswerTool` | Produces final summarized outputs |
| `UserInputTool` | Handles user input |
| `DuckDuckGoSearchTool` | Performs web search |
| `GoogleSearchTool` | Google search queries |
| `VisitWebpageTool` | Fetches and processes webpage data |

ü¶∏ *Example Usage:*

- Use `DuckDuckGoSearchTool` for party ideas
- Use `GoogleSearchTool` for caterers
- Use `PythonInterpreterTool` for seating calculations
- Use `FinalAnswerTool` to compile results

## üåê **Sharing & Importing Tools**

smolagents supports tool sharing and importing via Hugging Face and external ecosystems.

### üî∏ 1. **Sharing Tools**

Upload your custom tool to the **Hugging Face Hub**:

```python
party_theme_tool.push_to_hub("username/party_theme_tool", token="<HF_TOKEN>")

```

---

### üî∏ 2. **Importing Tools from Hub**

Load tools made by others:

```python
from smolagents import load_tool
image_generation_tool = load_tool("m-ric/text-to-image", trust_remote_code=True)

```

---

### üî∏ 3. **Importing Tools from Hugging Face Spaces**

```python
from smolagents import Tool
image_generation_tool = Tool.from_space(
    "black-forest-labs/FLUX.1-schnell",
    name="image_generator",
    description="Generate image from prompt"
)

```

*Requires `gradio_client` to connect to Spaces‚Äô backend.*

---

### üî∏ 4. **Importing LangChain Tools**

Reuse existing LangChain tools easily:

```python
from langchain.agents import load_tools
from smolagents import Tool

search_tool = Tool.from_langchain(load_tools(["serpapi"])[0])

```

---

### üî∏ 5. **Importing Tools from MCP Servers**

Connect to **Model Context Protocol (MCP)** servers to use external APIs or datasets.

```python
from smolagents import ToolCollection, CodeAgent
from mcp import StdioServerParameters

server_params = StdioServerParameters(command="uvx", args=["--quiet", "pubmedmcp@0.1.3"])
with ToolCollection.from_mcp(server_params, trust_remote_code=True) as tools:
    agent = CodeAgent(tools=[*tools.tools], model=InferenceClientModel())
    agent.run("Find a remedy for hangover.")

```
